HTML 5

Content Models

A normative description of what content must be included as children and descendants of the element. In simple terms "What it can contain"

Tag omission

A non-normative description of whether, in the text/html syntax, the start and end tags can be omitted.

Content attributes

A normative list of attributes that may be specified on the element (except where otherwise disallowed), along with non-normative descriptions of those attributes.

Allowed ARIA role attribute values

A normative list of ARIA role attribute values that may be specified on the element (except where otherwise disallowed). Each value is linked to a non normative description.

Allowed ARIA state and property attributes

Links to the Global aria-* attributes list and the allowed roles, states and properties table.

DOM interface

A normative definition of a DOM interface that such elements must implement.
example: interface HTMLParagraphElement: HTMLElement{}
Reference: https://www.w3.org/TR/html51/infrastructure.html#root-element
