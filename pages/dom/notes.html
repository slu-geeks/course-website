NOTES

Defer Attribute
  • Document tree should be loaded / fetch first
  • it can execute later
Async Attribute
  • does not wait for the document tree to fetch / load first
  • once it is available, it can now execute the script

Note: you cannot use both Defer and Async Attributes together
noscript
  • for browsers with no scripting support, or with scripting support disabled
window
  • global object
  • hook into the environment
navigator
  • represents the browser
document
  • hook into the actual document
  • owner of all other objects in your web page
body
  • attribute of the document element
Finding HTML Elements
     document.getElementById(id)
  • Find an element by element id
     document.getElementsByTagName(name)
  • Find elements by tag name
     document.getElementsByClassName(name)
  • Find elements by class name
Changing HTML Elements
     element.innerHTML = new html content
  • Change the inner HTML of an element
     element.attribute = new value
  • Change the attribute value of an HTML element
     element.setAttribute(attribute, value)
  • Change the attribute value of an HTML element
     element.style.property = new style
  • Change the style of an HTML element
Adding and Deleting Elements
     document.createElement(element)
  • Create an HTML element
     document.removeChild(element)
  • Remove an HTML element
     document.appendChild(element)
  • Add an HTML element
     document.replaceChild(element)
  • Replace an HTML element
     document.write(text)
  • Write into the HTML output stream
Properties
     innerText
  • Sets or retrieves the text between the start and end tags of the object.
  • valid for block elements only. By definition, elements that do not have both an opening and closing tag cannot have an innerText property.
     textContent
  • Sets or retrieves the text content of an object and any child objects.
  • preserves other white spaces
     innerHTML
  • parse whatever markup, possible to create new structures
     outerHTML
  • also gets the tags '<h1> Content </h1>'
     nodeName
  • returns the name of the specified node
  • If the node is an element node, the nodeName property will return the tag name.
  • If the node is an attribute node, the nodeName property will return the name of the attribute.
  • For other node types, the nodeName property will return different names for different node types.
     nodeValue
  • literal content
  • sets or returns the node value of the specified node
  • If the node is an element node, the nodeValue property will return null.
     nodeType
  • returns the node type, as a number, of the specified node.
  • If the node is an element node, the nodeType property will return 1.
  • If the node is an attribute node, the nodeType property will return 2.
  • If the node is a text node, the nodeType property will return 3.
  • If the node is a comment node, the nodeType property will return 8.
  • This property is read-only.
     children
  • returns a collection of an element's child elements, as an HTMLCollection object
     parentNode
  • returns the parent node of the specified node, as a Node object
  • returns null if you're already at the root of the tree
Methods
     querySelector
  • Returns the first Element within the document that matches the specified selector, or group of selectors.
     querySelectorAll
  • Returns all elements within the document that matches the specified selector, or group of selectors.
     matches
  • returns true if the element would be selected by the specified selector string; otherwise, returns false.
